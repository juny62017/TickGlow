# TicGlow

A cool and OG Tic Tac Toe game where you play against a simple(noob) bot opponent. hahaaaaa

## Features

- Play against a simple noo bot
- Neon themed and it geniunely looks cool
- friendly layout
- Real time results
- Win detection 
- Draw match detection
- Restart game button
- Highlighted winning boxes
- Smooth hover effects

## How To Use / Setup

- Click on my link
- and done there you can start by tapping on any box 
- Start playing against the bot
- Click restart to reset the game anytime
